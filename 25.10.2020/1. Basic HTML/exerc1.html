HTML

HTML(Hupertext Markup Language) is the set of markup sumbols or codes inserted in a file intended for display on a World Wide Web browser page. The markup tells the Web browser hot to display a Web page's words and images for the user. Each individual markup code is referred to as an element (but many people also refer to it as a tag). Some elements come in pairs that indicate when some display effect is to begin and when it is to end.
